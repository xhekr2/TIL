# Dart 기본
1. 출력  
- 콘솔에 데이터를 출력할 때는 기본적으로 `print()`함수를 사용한다. 실행될 때마다 자동으로 마지막에 줄 바꿈을 포함하기 때문에 세밀한 제어가 필요할 때는 `dart: io`패키지의 `stdout.write()`를 사용하기도 한다.
```dart
void main(){
  print('hello world');
}
```
2. 변수
    1. 변수 선언
        - Dart에서 기본적인 변수 선언 방법은 `var 변수명 = 값;`이다. var 데이터타입으로 선언된 변수는 값이 할당될 때 값의 데이터 타입을 자동으로 파악해 변수의 데이터 타입을 선언한다.
          ```dart
          void main(){
            var name = 'xhekr2';
            print(name);
          }
          ```
       - 정수 int(integer), 실수 double(double), 불 bool(Boolean), 글자 Srting(String), 다이나믹 dynamic(dynamic)으로도 변수를 선언할 수 있다.
         ```dart
         void main(){
           int a = 1;
         
           double b = 2;
         
           bool c = true;
         
           String d = hello;
         
           dynamic e = 값;
         }
         ```
