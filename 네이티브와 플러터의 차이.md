# 플러터(Flutter)란 무엇인가?
**플러터**는 Google이 만든 **오픈소스 UI 소프트웨어 개발 키트**(SDK)로,  
**하나의 코드베이스(Dart언어 사용)로 다양한 플랫폼에서 동작하는 앱을 빠르게 개발할 수 있는 크로스 플랫폼 프레임워크**.  

-**주요 특징**: **고성능 렌더링 엔진(Skia/Impeller)을 사용**하여 네이티브에 가까운 부드러운 화면 전환과 UI를 제공함.
-**핵심 언어**: 구글의 **Dart** 언어를 사용하여 쉽고 빠르게 개발할 수 있으며, **‘Hot Reload’(핫 리로드) 기능을 통해 코드를 수정하면 즉시 화면에 반영되는 빠른 개발 속도가 장점

## 네이티브와의 차이점
| 구분 | 플러터 | 네이티브 |
| --- | --- | --- |
| 개발 언어 | Dart | Kotlin, Swift 등 |
| 코드 | 하나로 여러 플랫폼 | 플랫폼마다 따로 |
| UI | Flutter 위젯 사용 | OS 기본 UI 사용 |
